# Early Prediction of Diabetes using KNN

This project is a Machine Learning web application that predicts whether a person is diabetic or not based on medical input parameters. The model is built using the K-Nearest Neighbors (KNN) algorithm and a simple user interface is developed using Streamlit.

## Objective

The aim of this project is to help in early detection of diabetes by analyzing patient data such as glucose level, BMI, age, and other health indicators.

## Technologies Used

Python, Pandas, NumPy, Scikit-learn, Streamlit

## Dataset

The project uses the PIMA Indians Diabetes Dataset which includes features like pregnancies, glucose level, blood pressure, BMI, age, and outcome.

## Model

K-Nearest Neighbors (KNN) algorithm is used. Data is preprocessed using standard scaling and split into training and testing sets.

## How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Train the model:
   python train_model.py

3. Run the app:
   python -m streamlit run app.py

## Output

The system predicts whether the person has a high or low risk of diabetes based on the input data.
